Coleta de Dados para Fins Educativos

Este portfólio coleta e exibe dados do seu navegador para demonstrar, de forma educativa, as técnicas de identificação digital usadas por plataformas de advertising, sistemas antifraude e ferramentas de segurança.

  • Canvas, Audio e WebGL fingerprints serão gerados e exibidos
  • User-Agent, SO, resolução, GPU e timezone serão coletados
  • Detecção de AdBlock, modo incógnito e automação será executada
  • Fontes do sistema serão enumeradas via técnica de medição CSS
  • WebRTC poderá revelar IPs da sua rede local
  • Detecção de dispositivos de mídia (câmera/microfone) sem permissão
  • Busca de presença em redes sociais via técnicas de timing
  • Nenhum dado é transmitido a servidores externos
  • Todo processamento ocorre localmente no seu navegador
Finalidade exclusivamente educativa. Os dados exibidos ilustram o que qualquer site na internet pode coletar sobre você hoje. O objetivo é gerar consciência sobre privacidade digital e demonstrar técnicas que analistas de segurança precisam conhecer para proteger usuários.

QUERY ARSENAL — KQL SENTINEL
Microsoft Sentinel — Log Analytics — KQL Query
// ── IMPOSSIBLE TRAVEL DETECTION ──────────────────────────────────────
// Detecta autenticação em dois países em janela de tempo impossível
SigninLogs
| where TimeGenerated > ago(1h)
| where ResultType == "0" // Sucesso
| extend Country = tostring(LocationDetails.countryOrRegion)
| summarize
    Countries = make_set(Country),
    LoginTimes = make_list(TimeGenerated),
    Count = count()
  by UserPrincipalName, bin(TimeGenerated, 1h)
| where array_length(Countries) > 1
| extend Severity = "High", TTP = "T1078 - Valid Accounts"
| project UserPrincipalName, Countries, LoginTimes, TTP, Severity

// ── POWERSHELL ENCODED COMMAND DETECTION ──────────────────────────────
// T1059.001 — PowerShell com payload ofuscado
DeviceProcessEvents
| where FileName =~ "powershell.exe"
| where ProcessCommandLine has_any ("-enc", "-EncodedCommand", "-ec")
| where InitiatingProcessFileName in~ ("winword.exe", "excel.exe", "outlook.exe")
| extend RiskScore = 95, TTP = "T1566.001 + T1059.001"
| project DeviceName, AccountName, ProcessCommandLine, InitiatingProcessFileName, TTP

// ── DNS TUNNELING DETECTION ────────────────────────────────────────────
// T1071.004 — Exfiltração via DNS queries longas
DnsEvents
| where strlen(Name) > 50
| where QueryType == "TXT" // TXT records são comuns em tunneling
| summarize count() by Computer, bin(TimeGenerated, 5m), Name
| where count_ > 20 // Volume anormal
| extend TTP = "T1071.004 - DNS Tunneling"

ENTROPIA POR SINAL — CONTRIBUIÇÃO PARA UNICIDADE

Entropia estimada em bits por sinal. Quanto maior, mais esse sinal contribui para distinguir seu navegador de todos os outros. Combinados, esses sinais criam um identificador com entropia total de ~30-40 bits — capaz de distinguir 1 bilhão a 1 trilhão de perfis únicos.
